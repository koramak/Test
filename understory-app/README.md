# Understory

A flashcard application for learning NYC-area wildflowers.

## Features

- **15 native wildflower species** with photos from Wikimedia Commons
- **Three study modes:**
  - **Learn** - View all information about each flower
  - **Practice** - Guess with hints available
  - **Test** - Identify flowers without hints
- **Progress tracking** - Mark flowers as mastered
- **Multiple photos** per species (tap to cycle)
- **Graceful fallbacks** for unavailable images

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Building for Production

```bash
npm run build
npm run preview
```
